## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.
```
use Node V18
create config.ts file in root of the project from config-sample.ts
put your dataBase credentials in it
```


## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
